## Build and run

**Build:**

```sh
git clone https://github.com/mogemimi/daily-snippets.git
cd daily-snippets/aligndiff

# Setup build tool
make -C ../nori bootstrap

# Build
make xcode

# Run
./bin/aligndiff -help
```
